2. Enumeration
#

FTP anonymous Login
#

In the scan we can see that you can log in via ftp with the anonymous user, which runs on port 21.

> ftp 10.10.11.247
Connected to 10.10.11.247.
220 (vsFTPd 3.0.3)
Name (10.10.11.247:mhil4ne): anonymous
230 Login successful.
Remote system type is UNIX.
Using binary mode to transfer files.
ftp>

After looking at all the files, we see something interesting in backup-OpenWrt-2023-07-26.tar.
